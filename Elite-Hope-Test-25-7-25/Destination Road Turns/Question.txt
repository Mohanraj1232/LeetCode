Destination Road Turns

In a city, the roads are parallel or perpendicular to each other. As maintenance work is going on certain road stretches are blocked. The program must print the minimum number of turns a motorist must make to reach the destination. If the destination cannot be reached, print -1.

Example Input/Output 1:
Input:
4 6
0 1 0 0 0 0
S 0 0 0 1 0
1 1 1 1 0 0
D 0 0 0 0 1

Output:
6

Explanation:
1 indicates a block.
